> Since v1.1.0

你可以通过 [`setPinnedTopRows`](../../api/api/02-row-api.md#setpinnedtoprows) 设置固定在顶部的行，通过 [`setPinnedBottomRows`](../../api/api/02-row-api.md#setpinnedbottomrows) 设置固定在底部的行。

你也可以通过 [`appendPinnedTopRows`](../../api/api/02-row-api.md#appendpinnedtoprows) 追加新的固定在顶部的行，通过 [`appendPinnedBottomRows`](../../api/api/02-row-api.md#appendpinnedbottomrows) 追加新的固定在底部的行。
