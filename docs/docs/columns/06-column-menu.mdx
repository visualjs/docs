在网格选项中可以通过 `getColumnMenuItems` 属性为列设置菜单。`getColumnMenuItems` 属性的值是一个回调函数，原型为：

```typescript
(params: GetColumnMenuItemsParams) => MenuItem[] | undefined;
```

`GetColumnMenuItemsParams` 是回调的参数：

```typescript
export interface GetColumnMenuItemsParams {
    // the column that was clicked
    column: string;
    grid: Grid;
}
```

`MenuItem` 是菜单项：

```typescript
export interface MenuItem {
    // is it a dividing line
    separator?: boolean;
    // disabled menu items
    disabled?: boolean;
    // display name
    name?: string;
    // icon
    icon?: string;
    // callback after the menu item is clicked
    action?: () => void;
    subMenus?: MenuItem[];
}
```
