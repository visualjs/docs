在初始设置列定义后，可以通过 `api` 更改列的定义。 本节介绍如何更新列定义。

## 添加/删除列

可以通过 `setColumns` 接口更新网格的列定义列表来实现添加和删除列。

## 修改列的状态

下列的几个接口可以用来修改列的状态：

- setColumnName
- setColumnPinned
- setColumnVisible
- setColumnWidth
- setColumnHeight

更多网格列相关的接口请看[列接口](../../api/api/03-column-api.md)小节。
